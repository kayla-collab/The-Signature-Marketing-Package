## ğŸ†˜ TROUBLESHOOTING

### **Client Can't Log In**
- Verify email and password are correct
- Check that account is marked "Active" in admin
- Ensure access hasn't expired
- Try clearing browser cache

### **Iframe Not Showing**
- Verify your live URL is correct in iframe src
- Check iframe height is set (min 800px)
- Ensure sandbox attributes are correct
- Test outside of Wix first

### **Files Won't Download**
- Check `allow-downloads` is in iframe sandbox
- Verify file URL is accessible
- Try right-click â†’ Save As

### **Videos Won't Play**
- Verify video URL is correct
- Check video is publicly accessible
- Try different browser
- Ensure `allow-popups` is in iframe sandbox
